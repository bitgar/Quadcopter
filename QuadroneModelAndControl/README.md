# Quadcopter

本项目致力于构建四旋翼无人机的动力学、运动学模型，在此基础上设计控制器，从而让飞机能正常平稳地飞行。

## 简介

笔者在航空航天领域待了十年，从一开始的无目的方向，到现在致力于成为无人机领域的专家，花费了太过漫长的时间。笔者想要融会贯通无人机建模、飞行控制、多源传感器融合导航三大领域。此项目就是笔者用于研究、梳理无人机建模与飞行控制的。

## 环境要求

- Simulink
